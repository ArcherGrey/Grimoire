# 项目总结

## 医疗信息管理系统

场景：页面加载执行操作时间优化提升 60%

思路：

结合 chrome perfomance 查看操作执行耗时分布

- JQuery
  - 尽可能使用 id 选择器
  - 在需要多次使用相同选择器的地方缓存，避免相同的对象重复查找
  - 循环调用 append 操作 dom 修改为用临时变量保存修改然后一次操作 dom
- 减少重排重绘
  - 直接用 JQuery 选择器多次修改样式 -> 将需要修改的样式合成一个 class 添加 class 即可
- http
  - 无需同步执行的请求 改为 异步执行
- js
  - 刪除冗余的 console

场景： websocket 实时通讯
